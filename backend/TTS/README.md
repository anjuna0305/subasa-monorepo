# Voc-Si: Speech-to-Speech Chatbot

## TTS Model

This guide will help you set up the TTS model for Voc-Si. Please ensure you have **Python 3.10.14** installed.

---

### Installation Instructions

1. **Create a Conda Environment**

   ```bash
   conda create -n vosci python=3.10.14
   
2. **Activate the Conda Environment**

   ```bash
   conda activate vosci
 
3. **Install Dependencies**

   ```bash
   python -m pip install -r path\to\requirements.txt
   
4. **Run the following command**

    ```bash
    pip install huggingface_hub transformers torch sounddevice numpy
